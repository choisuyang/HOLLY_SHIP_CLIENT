# hollyship-client

## 사람 중심의 음악 플랫폼 입니다.
## AI 대신 사용자가 음악을 추천해주는 아날로그 감성의 플랫폼 입니다

***

## 사용 스택

### Front 
* React-Native
* expo
* typscript
* react-native-kitten

### Back
* Express
* Typescript
* NodeJS
* Sequelize
* JWT
* AWS-EC2
* AWS-RDS
* AWS-S3
